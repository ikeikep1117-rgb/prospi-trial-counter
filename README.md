# 試練カウントツール

## 概要

プロスピAの特殊能力レベル上げで必要になる試練素材を、選手ごとに管理できるWebサイトです。選手名、特殊能力3つ、現在Lv、目標Lv、試練タイプ、所持素材を入力すると、必要数・所持数・不足数をまとめて確認できます。

## 主な機能

- 選手名の入力
- 選手ごとに特殊能力3つを登録
- 特殊能力ごとの現在Lv、目標Lv、試練タイプ入力
- 証、指南、極意書、閃きの印、開眼の印、真価の印の所持数カウンター
- 素材ごとのアイコン表示
- タイプ別12素材と共通の印3素材、それぞれの画像を手元の写真に差し替え
- 差し替えた素材画像をブラウザ内に保存
- 必要数、所持数、不足数の自動計算
- 不足素材を「何が何個足りないか」で個別表示
- コイン不足の確認
- 選手カードの追加・削除
- 入力内容のブラウザ自動保存
- スマホ、タブレット、PC対応のレスポンシブデザイン
- favicon、アプリアイコン、OGP画像、PWA用manifestを同梱

## 使用技術

- HTML
- CSS
- JavaScript
- IndexedDB
- SVGアセット

## 起動方法

ローカルで確認する場合は、このフォルダの `index.html` をブラウザで開きます。

開発用サーバーで確認する場合:

```bash
npx serve .
```
